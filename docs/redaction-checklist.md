# Redaction Checklist

Before publishing screenshots, JSON, logs, traces, or markdown:

- [ ] No secrets
- [ ] No private keys
- [ ] No internal IPs or hostnames
- [ ] No private repository names unless already public
- [ ] No local filesystem paths
- [ ] No email addresses
- [ ] No customer/user data
- [ ] No OpenProject private tickets
- [ ] No Nextcloud private paths
- [ ] Claims are evidence-backed
- [ ] Limitations are stated
- [ ] Synthetic examples are marked as synthetic
- [ ] Redacted examples are marked as redacted
